Game Directions

Test your memory with the Light and Sound Memory Game!

The objective of the game is to correctly repeat a longer and longer sequence of signals.

To play, press the START button. The computer will give the first signal. Repeat the signal by pressing the same button. But be careful... for every correct guess the computer will extend the sequence by one signal, making the game harder to win!

The player who successfully repeats eight signal sequences wins the game.
